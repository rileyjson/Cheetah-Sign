Cheetah Flow - The primary program used by Accutech.
DocuSign - The current program used by Cheetah Flow to pass out official documents that will be signed by clients.
Cheetah Sign - The program being designed by the students. Is intended to eventually become an in-house replacement for DocuSign to be used by Cheetah Flow.

.NET - The coding language that will be used to construct the backend of Cheetah Sign.
Vue.js - The coding language that will be used to construct the frontend of Cheetah Sign.
PostgreSQL - The database that will be used to hold onto the information of Cheetah Sign.
